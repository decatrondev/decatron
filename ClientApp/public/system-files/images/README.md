# Archivos de Imagen del Sistema

Esta carpeta contiene archivos de imagen predeterminados que los usuarios pueden seleccionar para sus Sound Alerts.

## Formatos Aceptados
- PNG (.png)
- JPG/JPEG (.jpg, .jpeg)

## Tamaño Máximo
- 5 MB por archivo

## Uso
Estas imágenes pueden usarse de dos formas:
1. Como imagen principal de una alerta (fileType='image')
2. Como imagen asociada a un archivo de audio para visualización

## Archivos de Ejemplo

Coloca aquí archivos de imagen de ejemplo que los usuarios puedan usar. Por ejemplo:
- `fbi.jpg` - Logo FBI
- `pepega.png` - Emote Pepega
- `pog.png` - Emote PogChamp
- `kappa.png` - Emote Kappa
- `hype.gif` - Animación de hype
- etc.

Recomendación: Usa imágenes cuadradas (200x200 o 400x400) para mejor visualización.

Los archivos colocados aquí estarán disponibles para todos los usuarios del sistema.
