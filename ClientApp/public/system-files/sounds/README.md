# Archivos de Sonido del Sistema

Esta carpeta contiene archivos de audio predeterminados que los usuarios pueden seleccionar para sus Sound Alerts.

## Formatos Aceptados
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)

## Tamaño Máximo
- 10 MB por archivo

## Archivos de Ejemplo

Coloca aquí archivos de audio de ejemplo que los usuarios puedan usar. Por ejemplo:
- `fbi.mp3` - Sonido de sirena FBI
- `vine-boom.mp3` - Efecto de sonido Vine Boom
- `bruh.mp3` - Efecto de sonido "Bruh"
- `discord-notification.mp3` - Sonido de notificación Discord
- etc.

Los archivos colocados aquí estarán disponibles para todos los usuarios del sistema.
