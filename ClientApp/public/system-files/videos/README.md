# Archivos de Video del Sistema

Esta carpeta contiene archivos de video predeterminados que los usuarios pueden seleccionar para sus Sound Alerts.

## Formatos Aceptados
- MP4 (.mp4)
- WEBM (.webm)

## Tamaño Máximo
- 50 MB por archivo

## Duración Máxima
- 30 segundos

## Archivos de Ejemplo

Coloca aquí archivos de video de ejemplo que los usuarios puedan usar. Por ejemplo:
- `fbi.mp4` - Video de sirena FBI
- `explosion.mp4` - Efecto de explosión
- `celebration.mp4` - Celebración animada
- `surprise.mp4` - Efecto de sorpresa
- etc.

Los archivos colocados aquí estarán disponibles para todos los usuarios del sistema.
